# Barqq Company Booking

A professional cleaning service booking platform for **Barqq Company** built with React. This project allows users to browse services, view company details, and book cleaning appointments online.

---

## 🌐 Live Demo

[Barqq Company Booking Live](https://clean-park.vercel.app/booking)

---

## 🛠️ Features

- **Service Booking**: Users can select and book cleaning services online.
- **Responsive Design**: Fully responsive layout for desktop, tablet, and mobile.
- **Navigation**: Smooth navigation through sections such as Home, About Us, Services, Branches, Features, Articles, and Request Service.
- **Interactive Forms**: User-friendly forms to submit booking requests.
- **React Router Integration**: Handles multiple pages and internal navigation smoothly.
- **Clean UI/UX**: Simple and modern design for a professional look.
- **Contact & Social Links**: Integrated contact information and social media links.

---

## 💻 Technologies Used

- **Frontend & Web Development**: HTML5, CSS3, JavaScript (ES6+), React.js, React Router, Bootstrap
- **State Management**: React Hooks
- **Tools & Productivity**: Git, GitHub
- **Deployment**: Vercel
